# sway-home

These are the dotfiles I use on most workstations. It supports the
following installation methods:

 * [setup.sh](FEDORA.md) (imperative config on any Linux distro *without* Nix).
 * [home-manager](HOME_MANAGER.md) (declarative config on any Linux distro *with* Nix).
 * [NixOS](NIXOS.md) (fully declarative operating system and home manager config).
